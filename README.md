# Gonzago Launcher

Лаунчер для GonzagoGL, который позволяет запускать прототип в разных режимах. Также он автоматически загружает прототип и патчит для работы fly/swim режима.

![preview](preview.png "Внешний вид лаунчера")

## В случае ошибки загрузки

Иногда по какой-то причине скачивание GonzagoGL не происходит. Пока не уверен из-за чего это происходит, но возможно из-за какого-то конфликта с zapret. В случай ошибки рекомендуется временно отключить zapret, либо использовать VPN, либо попробовать вручную скачать прототип от [сюда](http://www.spore.com/static/war/images/community/prototypes/gonzago.zip).

### Ручная распаковка Gonzago

Если вы скачали GonzagoGL вручную, создайте в папке с лаунчером папку `Gonzago` и распакуйте туда файлы из скаченного zip-архива 

## Как происходит патч

Лаунчер копирует оригинальный exe-шник и заменяет байт по адресу `0x155e8` на значение `0xB8`. Также он редактирует файл `Data/Gonzago.ini` и убирает параметр `load_game = predators`.
